# CPQ 系统设计文档

本目录是一套中文 CPQ 系统设计方案，适合用于领导汇报、需求评审、开发拆解和后续实施。

CPQ 是 Configure、Price、Quote 的缩写，即“配置、定价、报价”。系统目标是帮助销售团队快速选择产品配置、自动计算价格、控制折扣审批、生成正式报价单，并保留完整版本和审批记录。

## 建议第一版范围

第一版不要做成特别复杂的大平台，建议先交付一个能真实跑通业务的 MVP：

- 登录和角色权限
- 客户与联系人管理
- 产品、选项、套餐管理
- 价格本和折扣管理
- 新建报价与报价明细
- 后端自动计算报价
- 折扣和毛利审批
- 报价版本记录
- PDF 或 Excel 报价单导出
- 基础操作日志

复杂规则引擎、CRM/ERP 集成、多币种、渠道报价、电子签章等功能建议放到第二阶段或第三阶段。
