# CPQ 数据模型设计

## 1. 实体关系概览

```text
用户 -> 角色 -> 权限

客户 -> 联系人
客户 -> 商机 -> 报价

产品分类 -> 产品
产品 -> 产品选项 -> 选项值
产品 -> 套餐明细 -> 子产品
产品 -> 配置规则

价格本 -> 价格明细 -> 产品
折扣规则
税率规则

报价 -> 报价明细
报价 -> 报价版本
报价 -> 审批单 -> 审批日志
报价 -> 附件
报价 -> 订单

操作日志
```

## 2. 基础表

### 2.1 users 用户表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | uuid | 主键 |
| username | varchar | 登录账号，唯一 |
| password_hash | varchar | 加密后的密码 |
| display_name | varchar | 显示名称 |
| email | varchar | 邮箱 |
| phone | varchar | 手机号 |
| department_id | uuid | 部门 ID，可选 |
| status | varchar | active, disabled |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |

### 2.2 roles 角色表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | uuid | 主键 |
| code | varchar | 角色编码，唯一 |
| name | varchar | 角色名称 |
| description | text | 说明 |

### 2.3 customers 客户表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | uuid | 主键 |

| name | varchar | 客户名称 |
| industry | varchar | 行业 |
| region | varchar | 区域 |
| grade | varchar | 客户等级 |
| tax_number | varchar | 税号 |
| billing_address | text | 开票地址 |
| owner_id | uuid | 客户负责人 |
| status | varchar | active, disabled |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |

### 2.4 contacts 联系人表

| 字段 | 类型 | 说明 |

### 2.5 opportunities 商机表
